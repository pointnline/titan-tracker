2024~현재
Microsoft 시총 $3T+ 돌파. Copilot+ PC 발표. AI 전 제품 통합 (Windows, Office, Teams). NVIDIA와 함께 AI 슈퍼컴퓨터 구축.
2023
OpenAI에 $10B+ 추가 투자. Bing AI (GPT-4 기반), Microsoft 365 Copilot 출시. Activision Blizzard $69B 인수 완료.
2021
Microsoft 회장 겸임. 시총 $2T 돌파. Nuance Communications $19.7B 인수 (의료 AI). Windows 11 출시.
2019
OpenAI에 $1B 투자 — AI 파트너십 시작. Azure 성장 가속, AWS와의 격차 축소. 시총 $1T 돌파.
2018
GitHub $7.5B 인수. 오픈소스 전략 대전환 — "Microsoft ❤️ Open Source". Azure DevOps 출시.
2016
LinkedIn $26.2B 인수 — 역대 최대 규모 인수. 전문가 네트워크 + 데이터 전략.
2014
Microsoft 3대 CEO 취임. "Cloud First, Mobile First" 전략 선언. 문화 변혁 시작. Hit Refresh.
2011
Server & Tools → Cloud & Enterprise 책임자. Azure를 핵심 사업으로 육성.
1992
Microsoft 입사. Windows NT, SQL Server, Azure 등 핵심 기술 리더십. 22년간 내부 성장.
1988
Sun Microsystems에서 경력 시작. 이후 UW Milwaukee CS 석사, UChicago Booth MBA.
1967
인도 하이데라바드 출생. 크리켓 선수 꿈 → 공학으로 전환. Manipal Institute of Technology 진학.
